All depandence are incomposer.json and package.json
to reinstall you can delete node_moduels and vendor and then run:
/* composer install */
/* npm install */

to migrate and refresh and seed Db
/* php artisan migrate:refresh --seed */

use /* npm run watch */ to load all vue file
